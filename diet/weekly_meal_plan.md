# 每周营养均衡计划（28岁男性 / 67kg / 维持体型版）

## 目标
- 蛋白质：约 100–130g / 天
- 主食：不过量，优先米饭、燕麦、土豆、全麦面
- 蔬菜：至少 500g / 天，深绿叶菜 + 菌菇 + 彩色蔬菜轮换
- 水果：1–2 份 / 天
- 油脂：控油，优先蒸、煮、炖、炒；每日烹调油尽量 20–30g 内
- 饮食风格：家常、容易买菜、容易自己做，参考徽式/老乡鸡系的朴素中式搭配思路

## 固定原则
1. 每餐尽量满足：`1份优质蛋白 + 1份蔬菜 + 1份主食`
2. 晚餐比午餐略轻，避免高油高糖夜宵
3. 一周至少：
   - 鸡胸/鸡腿 3 次
   - 牛肉/瘦猪肉 2 次
   - 鱼虾 2 次
   - 鸡蛋 5–7 个
   - 豆腐/豆制品 3 次
4. 早餐优先高蛋白，不用太复杂
5. 口味可适当偏 `江西家常风味`：允许加入 `小米椒 / 青红椒 / 剁椒 / 辣椒粉`，但仍以少油、不过咸为原则

## 采购节奏
- **周末主采购**：肉类、鸡蛋、豆制品、耐放蔬菜（如西兰花、胡萝卜、土豆、洋葱、冬瓜、南瓜）、水果一次多买一些
- **工作日补采购**：叶菜、菌菇、番茄、黄瓜等新鲜蔬菜少量补货
- 每日菜谱的“今日需购买食材”应优先按这个节奏判断：
  - 周六 / 周日默认提示 `主采购清单`
  - 其余工作日默认提示 `补货清单/缺什么买什么`

## 周计划

### 周一
- 早餐：`燕麦 + 水煮蛋2个 + 无糖酸奶 + 香蕉`
- 午餐：`番茄炖牛腩 + 清炒西兰花 + 米饭`
- 晚餐：`香菇蒸鸡腿 + 凉拌黄瓜 + 紫菜蛋花汤`

### 周二
- 早餐：`全麦面包 + 煎蛋2个 + 牛奶/豆浆 + 苹果`
- 午餐：`青椒鸡丁 + 蒜蓉生菜 + 米饭`
- 晚餐：`鲈鱼/鳕鱼清蒸 + 菠菜豆腐汤 + 小份红薯`

### 周三
- 早餐：`玉米 + 鸡蛋2个 + 无糖酸奶 + 橙子`
- 午餐：`土豆烧牛肉/瘦肉 + 清炒油麦菜 + 米饭`
- 晚餐：`虾仁滑蛋 + 清炒西葫芦 + 菌菇汤`

### 周四
- 早餐：`燕麦粥 + 茶叶蛋2个 + 梨`
- 午餐：`家常炖鸡块（可偏老乡鸡风格） + 青菜 + 米饭`
- 晚餐：`豆腐烧虾仁/肉末豆腐 + 清炒上海青 + 小米粥`

### 周五
- 早餐：`全麦吐司 + 花生酱薄涂 + 鸡蛋2个 + 牛奶`
- 午餐：`芹菜香干炒肉丝 + 番茄炒蛋 + 米饭`
- 晚餐：`冬瓜炖排骨/鸡胸 + 凉拌木耳 + 小份米饭`

### 周六
- 早餐：`山药粥 + 水煮蛋2个 + 苹果`
- 午餐：`洋葱牛肉 + 清炒菜心 + 米饭`
- 晚餐：`煎三文鱼/巴沙鱼 + 蒜蓉西兰花 + 南瓜`

### 周日
- 早餐：`杂粮粥 + 鸡蛋2个 + 酸奶`
- 午餐：`香菇板栗鸡/土豆鸡块 + 生菜 + 米饭`
- 晚餐：`番茄豆腐虾仁汤 + 凉拌菠菜 + 玉米`

## 基础采购清单（按一周）
### 蛋白质
- 鸡胸/鸡腿：`1.5–2kg`
- 牛肉/瘦猪肉：`800g–1kg`
- 鱼：`2–3条/份`
- 虾仁：`500g`
- 鸡蛋：`14–18个`
- 北豆腐/嫩豆腐：`3–4盒`
- 无糖酸奶：`5–7杯`
- 牛奶/无糖豆浆：`5–7份`

### 主食
- 大米
- 燕麦
- 全麦面包/吐司
- 红薯 / 南瓜 / 山药 / 玉米（任选轮换）

### 蔬菜
- 西兰花
- 菠菜 / 生菜 / 油麦菜 / 上海青 / 菜心（轮换）
- 黄瓜
- 番茄
- 青椒
- 土豆
- 西葫芦
- 冬瓜
- 木耳
- 香菇 / 菌菇类
- 洋葱

### 水果
- 香蕉
- 苹果
- 橙子 / 梨

### 常备调味
- 生抽、老抽、蚝油、盐、黑胡椒、醋、葱姜蒜

## 每日生成规则（给自动汇报用）
- 按当天星期输出对应的 `早餐 / 午餐 / 晚餐`
- 采购提醒遵循两套逻辑：
  - **周六 / 周日**：输出 `本周主采购`，按 3-5 天肉类/蛋白 + 耐放蔬菜 + 早餐食材 + 常见水果 来组织，不是只列当天菜单食材
  - **周一到周五**：输出 `今日补货`，只列需要补的新鲜叶菜、菌菇、番茄、黄瓜、水果等，以及临时缺口食材
- 默认不重复列出：`油、盐、生抽、老抽、蚝油、黑胡椒、葱姜蒜`
- 如果某道菜有可替代食材，优先给更容易买到的版本
- 输出风格：简洁、适合手机阅读
